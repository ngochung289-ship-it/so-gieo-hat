Connect your Firebase project to go live. Takes 3 minutes — everyone joins via a shared link, wishes sync in real time.

1
Go to console.firebase.google.com → Add project → name it anything (e.g. hr-magic-wand) → disable Google Analytics → Create project
2
Inside your project → click Realtime Database (left sidebar) → Create database → choose any region → select Start in test mode → Enable
3
Click the ⚙️ gear icon → Project settings → scroll to Your apps → click </> (Web) → register app → copy the databaseURL and apiKey values below
